# Open Source Community

Status: Planned.

Audience: contributors, maintainers, open source program reviewers.

Lumina should be open source, community-driven, and evidence-led.

## Community Principles

- Be clear about current status.
- Welcome documentation, examples, testing, and architecture contributions.
- Treat AI-assisted work as accountable work.
- Require evidence for benchmark, security, and compatibility claims.
- Use decision records for long-lived architecture decisions.
