# Product Strategy

Status: Planned.

Audience: maintainers, contributors, future website visitors, AI agents.

Lumina's product strategy is to make the framework itself the map of the application.

## Positioning

Lumina is an app-graph-native, SEO-first React framework for humans and AI agents.

## Strategic Wedge

- Semantic app graph.
- Agent-safe workflows.
- SEO-safe rendering.
- Bun-first adapter paths with runtime compatibility.
- Stable machine-readable manifests.

See `VISION.md`, `docs/risk-mitigation.md`, and `docs/prototype-acceptance.md`.
