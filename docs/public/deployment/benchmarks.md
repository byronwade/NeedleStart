# Benchmark Honesty

Status: Planned.

Audience: app developers, maintainers, performance reviewers.

NeedleStart should be fast, but public performance claims require evidence.

Every performance claim must link to:

- raw results
- fixture source
- runtime versions
- hardware or environment
- run count
- variance
- methodology
