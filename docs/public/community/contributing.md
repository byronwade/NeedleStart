# Contributing

Status: Planned.

Audience: contributors, AI agents.

Lumina is currently in Phase 1: monorepo scaffold. The first contributor responsibility is to keep direction, architecture, implementation tasks, and docs coherent while the prototype foundation is built.

## Contributor Path

1. Read project status.
2. Read AGENTS.
3. Read relevant docs.
4. Use task and ADR templates when needed.
5. Keep docs current with behavior.
6. Run available checks.
7. State checks that cannot run.
