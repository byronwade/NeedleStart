# Community

Status: Planned.

Audience: contributors, maintainers, open source program reviewers.

Lumina should be open source, evidence-led, and welcoming to both human and AI-assisted contributors.

## Community Principles

- Be clear about what exists.
- Keep planned behavior separate from implemented behavior.
- Review AI-assisted work with the same standard as human work.
- Require evidence for benchmark, security, and compatibility claims.
- Use decision records for long-lived architecture choices.
