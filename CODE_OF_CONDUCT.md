# Code of Conduct

NeedleStart is an open source project for humans and AI agents. The community should be direct, respectful, evidence-driven, and safe for contributors with different backgrounds and experience levels.

## Expected Behavior

- Focus criticism on ideas, code, docs, and evidence.
- Assume good faith unless there is clear evidence otherwise.
- Be specific when reporting bugs, documentation gaps, or design risks.
- Respect maintainer decisions about scope, safety, and release timing.
- Keep automated or AI-assisted contributions accountable to a human reviewer.

## Unacceptable Behavior

- Harassment, threats, discriminatory language, or personal attacks.
- Publishing private information without permission.
- Repeatedly derailing technical discussions after maintainers redirect the topic.
- Misrepresenting planned behavior as implemented behavior.
- Using automation to spam issues, pull requests, discussions, or maintainers.

## Reporting

Until a dedicated conduct committee exists, report conduct issues privately to the project maintainers. Maintainers should acknowledge reports, preserve relevant evidence, avoid public escalation when possible, and take proportionate action.

## Enforcement

Possible actions include clarification, moderation, temporary restrictions, or removal from project spaces. Enforcement decisions should prioritize contributor safety, project health, and documented evidence.
